
Overview
--------
The company_subuser_dashboard.module is a framework which provides the functionalities for users having subuser role. 


Requirements
------------
Drupal 7.x
PHP 5.1
MySQL 5

This is the necessary module to be installed to make the site functional.


Features
--------
User features:
 - This module will design the dashboard.
 - Provide facility for subuser to add strong tags to the list of videos.
 - Provide facility to do translation job to submitted videos.
